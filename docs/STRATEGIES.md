# 量化策略說明文件

> 本文件說明 `twquant` 平台所有已驗證量化策略的數學公式、適用範圍與參數。  
> 所有策略均為原創組合，未使用網路爬取策略，進場/出場條件均以數學不等式定義。  
> **策略審查標準**：最佳標的的超額報酬（相對 0050 買進持有）必須 > 0，否則刪除。

---

## 策略分類說明

| 標籤 | 意義 |
|------|------|
| [全市場] | 適用所有具備足夠歷史資料的股票（至少 120 交易日）|
| [強勢股] | 要求趨勢確認條件，僅適用已展現動能的股票 |
| [突破型] | 以價量突破事件為觸發，需量能支撐避免假突破 |

MA5(t) < MA20(t)                   # 短線均線下穿中線

### M｜風險調整動能 RAM

**分類**: [全市場]（偏向景氣循環股：半導體、DRAM、科技）  
**函數**: `strategy_risk_adj_momentum`

**核心公式：RAM（Risk-Adjusted Momentum）**：
```
ret₂₀  = (Pt / Pt-20) - 1                          # 20日報酬
σ₂₀    = std(daily_returns, 20) × √20              # 20日標準化波動率
RAM    = ret₂₀ / σ₂₀                               # Sharpe-like 動能因子
```

> RAM 類似 Sharpe Ratio 的動能版：同樣的報酬下，波動率越低分數越高。  
> 高 RAM 代表「相對波動率的強勢動能」，可有效過濾假突破的高波動股。

**進場條件（同時滿足）**：
```
RAM    > 0.7                        # 風險調整後動能充足
Pt    > MA60(60)                   # 趨勢確認
MA60(60) > MA120(120)              # 中長期趨勢向上（雙均線過濾）
```

**出場條件（任一滿足）**：
```
RAM    < 0.0                        # 動能轉負
Pt    < MA60(60) × 0.97            # 跌破中期趨勢3%停損
```

**參數**:
| 參數 | 值 | 意義 |
|------|-----|------|
| `ret_window` | 20 | 動能計算窗口 |
| `vol_window` | 20 | 波動率計算窗口 |
| `ram_entry` | 0.7 | RAM進場門檻 |
| `ram_exit` | 0.0 | RAM出場門檻（動能歸零） |
| `ma_trend` | 60 | 中期趨勢均線 |
| `ma_long` | 120 | 長期趨勢均線 |
| `stop_buffer` | 3% | MA60之下緩衝帶 |

**實證驗證（2022-2026，含交易成本）**：
- 南亞科（2408）：**超額報酬 +285%（所有策略最高）**，Sharpe 1.74
- 最適合景氣循環股（高 Beta 品種）

### N｜唐奇安通道突破

**分類**: [突破型]  
**函數**: `strategy_donchian_breakout`

**核心公式：唐奇安通道（Donchian Channel）**：
```
DC_upper(N, t) = max(High_t-1, High_t-2, ..., High_t-N)   # N日最高點（昨日以前）
DC_lower(N, t) = min(Low_t-1,  Low_t-2,  ..., Low_t-N)    # N日最低點
```

> 與策略 H 的差異：H 用收盤價創高，N 用 High 價（含盤中）定義通道，  
> 通道具有方向性，下軌也作為出場依據，邏輯更完整。

**進場條件（同時滿足）**：
```
Pt     > DC_upper(20, t-1)         # 突破前期20日通道上軌（昨日計算）
Vol_t  > Vol₂₀_mean × 1.2         # 量能確認（避免假突破）
Pt     > MA60(60)                  # 趨勢向上
RSI₁₄  < 76                       # 未超買
```

**出場條件（任一滿足）**：
```
Pt     < DC_lower(20, t)           # 跌破通道下軌
RSI₁₄  > 85                       # 極度超買出場
Pt     < MA60(60) × 0.95          # 長期趨勢破壞停損
```

**參數**:
| 參數 | 值 | 意義 |
|------|-----|------|
| `dc_window` | 20 | 唐奇安通道窗口（天） |
| `vol_ratio` | 1.2× | 量能確認門檻 |
| `ma_filter` | 60 | 趨勢過濾均線 |
| `rsi_entry` | 76 | RSI進場上限 |
| `rsi_exit` | 85 | RSI超買出場線 |
| `stop_buffer` | 5% | MA60之下停損帶 |

**實證驗證（2022-2026，含交易成本）**：
- 台達電（2308）：超額報酬 +201%，Sharpe 1.72

### O｜Fibonacci EMA絲帶

**分類**: [動能科技股]（半導體、IC設計、電子零件；弱勢股不適用）  
**函數**: `strategy_ema_ribbon`

**核心公式：費氏 EMA 排列分數**：
```
EMA_n = 指數移動平均，span = n（費氏數列：8, 13, 21, 34, 55, 89）

conf(t) = (EMA₈ > EMA₁₃) + (EMA₁₃ > EMA₂₁) + (EMA₂₁ > EMA₃₄)
        + (EMA₃₄ > EMA₅₅) + (EMA₅₅ > EMA₈₉)

        範圍：0 ~ 5（5 = 所有相鄰EMA對均為正排列 = 完整多頭）
```

> 費氏數列（Fibonacci sequence: 8, 13, 21, 34, 55, 89）在技術分析中被廣泛認為對應市場週期。  
> conf=5 表示從最短期到最長期所有時間框架同時看多，是多頭趨勢最強確認點。  
> conf 剛從 4→5 為最佳進場時機（趨勢剛完成確立的第一天）。

**進場條件（同時滿足）**：
```
conf(t) = 5                        # 完整費氏多頭排列
conf(t-1) < 5                      # 昨日尚未完整（今日才剛確立）
Pt > EMA₈₉                        # 股價在最長期EMA之上
RSI₁₄ < 72                        # 未超買
```

**出場條件（任一滿足）**：
```
conf(t) < 3                        # 多頭排列鬆動（三對以上排列失效）
Pt < EMA₅₅ × 0.97                 # 跌破中期EMA55下方3%
```

**參數**:
| 參數 | 值 | 意義 |
|------|-----|------|
| `ema_seq` | [8,13,21,34,55,89] | 費氏數列EMA組合 |
| `entry_conf` | 5 | 進場時需達到的完整排列分數 |
| `exit_conf` | 3 | 出場時的排列分數下限 |
| `rsi_cap` | 72 | 進場RSI上限 |
| `ema55_buffer` | 3% | EMA55下方停損緩衝 |

**實證驗證（2022-2026，含交易成本，科技股42支）**：
- 致茂電子（2360）：+534.9%，**超額 +387.6%（平台所有策略最高）**，Sharpe 1.65
- 京元電子（2449）：+317.9%，超額 +170.6%，Sharpe 1.46
- 欣興（3037）：+277.4%，超額 +130.2%，Sharpe 1.28
- 台達電（2308）：+255.9%，超額 +108.6%，Sharpe 1.43
- 日月光投控（3711）：+232.1%，超額 +84.8%，Sharpe 1.35
- 正超額比率：11/42 支（26%），建議配合選股篩選器使用

### P｜複合趨勢分數 CTS

**分類**: [全市場]（對高 β 科技股效果最佳）  
**函數**: `strategy_composite_trend_score`

**核心公式：Composite Trend Score (CTS)**：
```
trend_signal = [(MA5 > MA20) + (MA20 > MA60)] / 2      # 0, 0.5, 或 1.0

momentum_sig = +1.0   if ret₂₀ > 5%                    # 強勢動能
             =  0.0   if -3% < ret₂₀ ≤ 5%              # 中性
             = -1.0   if ret₂₀ ≤ -3%                   # 弱勢動能

volume_signal = 1.0   if Vol₅_mean / Vol₂₀_mean > 1.3x
              = 0.0   otherwise

CTS = 0.4 × trend_signal + 0.4 × momentum_sig + 0.2 × volume_signal
```

> CTS 是量化分析師的三因子加權評分模型（趨勢40%+動能40%+量能20%）。  
> CTS > 0.6 代表趨勢+動能雙確認，量能輔助，觸發進場。  
> 指標從 0.5 → 0.6 的穿越（threshold crossing）是訊號點，非持倉訊號。

**進場條件（同時滿足）**：
```
CTS(t) > 0.6                       # 複合分數剛超過門檻
CTS(t-1) ≤ 0.6                     # 昨日尚未超過（今日才穿越）
RSI₁₄ < 68                        # 未超買
```

**出場條件（任一滿足）**：
```
CTS < 0.1                          # 三因子全面轉弱
Close < MA60(60) × 0.95            # 跌破長期趨勢5%停損
```

**參數**:
| 參數 | 值 | 意義 |
|------|-----|------|
| `w_trend` | 0.4 | 趨勢因子權重 |
| `w_momentum` | 0.4 | 動能因子權重 |
| `w_volume` | 0.2 | 量能因子權重 |
| `entry_threshold` | 0.6 | 進場CTS門檻 |
| `exit_threshold` | 0.1 | 出場CTS門檻 |
| `ret_strong` | 5% | 動能強勢門檻 |
| `ret_weak` | -3% | 動能弱勢門檻 |
| `vol_ratio` | 1.3× | 量能確認門檻 |
| `rsi_cap` | 68 | 進場RSI上限 |
| `stop_buffer` | 5% | MA60之下停損帶 |

**實證驗證（2022-2026，含交易成本，全宇宙112支）**：
- 京元電子（2449）：+454.2%，**超額 +307.0%，Sharpe 1.75（策略最高 Sharpe）**
- 致茂電子（2360）：+384.9%，超額 +237.7%，Sharpe 1.47
- 欣興（3037）：+371.6%，超額 +224.3%，Sharpe 1.36

### Q｜多周期動能共振

**分類**: [全市場]（對高 β 半導體/DRAM/PCB 股票特別有效）  
**函數**: `strategy_multi_period_momentum`

**核心公式：三時間框架動能同步**：
```
ret₅  = (Pt / Pt-5) - 1            # 5日動能（極短期週線代理）
ret₂₀ = (Pt / Pt-20) - 1           # 20日動能（月線代理）
ret₆₀ = (Pt / Pt-60) - 1           # 60日動能（季線代理）

confluence(t) = [ret₅ > 0.5%] AND [ret₂₀ > 3%] AND [ret₆₀ > 8%]
```

> 三周期動能同時為正是罕見的強力信號，代表從超短期到中長期的完整趨勢共振。  
> 使用「剛共振的第一天」（confluence 從 False → True）避免追高。  
> ret₆₀ > 8% 確保長期趨勢方向，ret₅ > 0.5% 確認近期未反轉。

**進場條件（同時滿足）**：
```
ret₅  > 0.5%                        # 5日小正動能（近期未反轉）
ret₂₀ > 3%                          # 20日中期動能充足
ret₆₀ > 8%                          # 60日長期趨勢確認
confluence(t-1) = False             # 昨日非三期共振（今日才首次共振）
RSI₁₄ < 70                         # 未超買
```

**出場條件（任一滿足）**：
```
ret₂₀ < -3%                         # 中期動能轉負
Close < MA60(60) × 0.95             # 長期趨勢破壞停損
```

**參數**:
| 參數 | 值 | 意義 |
|------|-----|------|
| `ret5_min` | 0.5% | 極短期動能下限 |
| `ret20_min` | 3% | 中期動能下限 |
| `ret60_min` | 8% | 長期動能下限 |
| `rsi_cap` | 70 | 進場RSI上限 |
| `exit_ret20` | -3% | 中期動能出場線 |
| `stop_buffer` | 5% | MA60之下停損帶 |

## 指標公式速查

所有指標均在 `src/twquant/indicators/basic.py` 中實作：

```
MA(n)      = 簡單移動平均，窗口 n 天
EMA(n)     = 指數移動平均，span=n
RSI(n)     = Wilder 平滑法，Avg Gain / Avg Loss，期間 n 天
MACD       = EMA12 - EMA26，訊號線 EMA9
Bollinger  = MA20 ± 2σ
KD(9,3,3)  = RSV → EWM K → EWM D
ATR(14)    = max(H-L, |H-C₋₁|, |L-C₋₁|) 的 14 日均值
BIAS(20)   = (Close - MA20) / MA20 × 100
Williams %R(14) = (HH14 - Close) / (HH14 - LL14) × -100
OBV        = 累計（價漲+量，價跌-量）
StochRSI   = 對 RSI 序列再做 KD 處理
VWAP(20)   = rolling 20 日 (典型價 × 成交量) / 成交量
Donchian(20) = 20日 High 最高 / Low 最低 / 中線
RAM(20)    = ret₂₀ / (σ₂₀ × √20)  [本平台原創因子]
```

## 回測環境

- 期間：2022-01-01 ～ 2026-04-30（~4.3 年）
- 初始資金：TWD 1,000,000
- 交易成本：手續費 0.1425% × 0.6折（買賣各一次） + 證交稅 0.3%（賣出）
- 滑價：0.1%
- 回測引擎：VectorBT（`src/twquant/backtest/engine.py`）
- 基準：0050 買進持有（同期 +146%，Sharpe 1.29）
