🎯 系統總覽 — 平台能做什麼

TWQuant 是專為台灣股市設計的量化交易研究與選股平台。 它整合了「資料 → 策略 → 回測 → 訊號 → 推播」的完整鏈路， 並依台股實際交易規則（手續費、證交稅、漲跌幅、零股）內建到回測引擎中。

📥

資料管線

FinMind API + 本地 CSV / SQLite / ArcticDB。自動分割還原、合理性檢查、HWM 斷點續傳。

🧠

策略註冊

9 種內建策略，5 種已實證超額報酬 > 0。可擴充 Rust 自訂策略（PyO3 橋接）。

🔬

VectorBT 回測

整合台股手續費 0.1425%×折扣、證交稅 0.3%、滑價 0.1%，支援追蹤停損。

📡

訊號掃描

全宇宙 × 多策略當日進場掃描，依板塊分組顯示。共用 scan_universe()。

🔔

告警系統

突破 N 日高低 / RSI 穿越 / 策略訊號規則，背景 worker + 手動掃描 + 已讀管理。

每日選股

訂閱策略 → 盤後自動掃描 → Discord 推播 + 30 日歷史檢索。

關鍵步驟說明

FinMind 抓取 + 速率限制

登入 token 可用 600 calls/hr，未登入 300 calls/hr。FinMindProvider 內建滑動視窗速率限制器，自動 sleep 防止 429。

Sanity Check + Split Adjust

對 OHLCV 做合理性過濾（high ≥ low、volume ≥ 0），偵測除權息日，並自動向後還原分割（如 1:5 拆股 → 還原前期股價）。

HWM 寫入 SQLite

upsert(symbol, df) 以 date 為主鍵冪等寫入。get_hwm() 回傳最後資料日，供下次接續同步使用。

策略產生進出場訊號

generate_signals(df) → (entries, exits) 回傳 bool 陣列。所有策略繼承 BaseStrategy 抽象類。

VectorBT 套用台股成本

TWSEBacktestEngine.run() 套用手續費 0.1425% × 折扣、滑價 0.1%，可選追蹤停損。回傳 14 項績效指標。

Dashboard 或 Discord 輸出

Streamlit 顯示 K 線、資金曲線、卡片指標；或推播每日選股清單到 Discord（每策略一張 embed）。

🚀 快速開始 — 3 步驟啟動

環境準備

安裝 Python 依賴與 Rust 工具鏈，編譯 Rust 模組。

下載種子資料

建議首次跑 --universe（49 支分析師宇宙），或 --all 全市場（需 Token）。

啟動 Dashboard

執行 make run → 開 localhost:8501 → 完成 onboarding。

# 1. 安裝
poetry install
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
make build-rust

# 2. 種子資料
poetry run python scripts/seed_data.py --universe

# 3. 啟動
make run
Docker 一鍵啟動： make docker-up，等同 docker compose up -d。
沒有 FinMind Token 也能跑： 系統會降級用 data/sample/ 中的範例 CSV，但只能看 4-5 支股票。 至 finmindtrade.com 免費註冊取得 Token 後即可全功能運作。

🧠 9 種內建策略

所有策略均為原創組合，進出場條件以數學不等式定義。 審查標準：最佳標的的超額報酬（相對 0050 買進持有）必須 > 0，否則刪除。 完整公式請見 docs/STRATEGIES.md。

已驗證生產策略（5 種）

F｜動能精選 ★ 強勢股 momentum_concentrate

20 日動能 > 5% 且收盤價 > MA60；跌破 MA60 × 0.97 出場。

進場：ret₂₀ > 5% AND Close > MA60 出場：Close < MA60 × 0.97
📊 台達電（2308）+369%，Sharpe 1.66，超額 +223%
H｜量價突破 突破型 volume_breakout

創 20 日新高 + 量比 > 1.5x + Close > MA60 + RSI < 76。

進場：Close > HIGH₂₀ AND Vol₅/Vol₂₀ > 1.5x AND Close > MA60 AND RSI < 76 出場：Close < MA60 × 0.96 OR RSI > 85
📊 台達電 Sharpe 1.81（所有策略最高），超額 +228%
L｜三線扭轉 全市場 triple_ma_twist

捕捉趨勢結構「剛確立」的最早入場點。多頭排列 t 為 True、t-1 為 False。

alignment = (MA5>MA20) AND (MA20>MA60) AND (P>MA5) 進場：alignment(t)=True AND alignment(t-1)=False AND RSI < 72
📊 台達電 +258%，超額 +111%（交易少但精確）
M｜風險調整動能 RAM 景氣循環股 risk_adj_momentum

原創 RAM 因子：ret₂₀ / (σ₂₀ × √20)，類似 Sharpe 的動能版。

RAM = ret₂₀ / (σ₂₀ × √20) 進場：RAM > 0.7 AND P > MA60 AND MA60 > MA120 出場：RAM < 0.0 OR P < MA60 × 0.97
📊 南亞科（2408）超額 +285%（所有策略最高），Sharpe 1.74
N｜唐奇安通道突破 突破型 donchian_breakout

用 High 價（含盤中）定義通道，下軌作為出場依據。

DC_upper = max(High_t-1, ..., High_t-N) 進場：P > DC_upper(20) AND Vol > Vol₂₀×1.2 AND P > MA60 AND RSI < 76 出場：P < DC_lower(20) OR RSI > 85
📊 台達電 超額 +201%，Sharpe 1.72

📊 資料管線運作機制

儲存抽象層

DataStorage 是抽象基底，提供統一介面，子類包含 SQLiteStorage（預設）與 ArcticDBStorage（正式環境）。 所有資料以 {dataset}/{stock_id} 命名（例 daily_price/2330）。 

storage.upsert(symbol, df)        # 冪等寫入，以 date 為主鍵
storage.load(symbol, start, end)  # 日期範圍讀取
storage.get_hwm(symbol)           # 最後資料日
storage.get_dates(symbol)         # 所有已存日期（給闕漏偵測）
storage.list_symbols()            # 列出所有 symbol

同步引擎

MarketDataSyncEngine 支援三種模式：

  • initial_full_sync：首次全市場下載，產生器形式 yield 進度。
  • incremental_sync：從 HWM + 1 天接續，關注清單優先。
  • detect_and_fill_gaps：偵測日期空洞並回補（合併連續區間減少 API 呼叫）。

背景自動同步

auto_sync.ensure_running() 啟動 daemon thread：

  • 盤中（09:00-13:30 週一至週五）每 30 分鐘同步一次
  • 盤後每小時同步一次
  • @st.cache_resource 確保只啟動一次

合理性檢查 + 分割還原

TWSEDataSanityChecker 過濾掉 OHLC 不合理的列（high < low 等）並隔離。 split_adjust 偵測除權息日，向後還原所有歷史價格，確保回測準確。

FinMind 速率限制：登入 Token = 600 calls/hr，未登入 = 300 calls/hr。 _RateLimiter 滑動視窗自動 sleep，無需手動處理 429。

🔬 VectorBT 回測引擎

主要介面

from twquant.backtest.engine import TWSEBacktestEngine
from twquant.strategy.registry import get_strategy

# 1. 取策略 → 產生訊號
strategy = get_strategy("momentum_concentrate")
entries, exits = strategy.generate_signals(df)

# 2. 跑回測
engine = TWSEBacktestEngine()
metrics = engine.run(
    price=df["close"],
    entries=entries, exits=exits,
    init_cash=1_000_000,
    broker_discount=0.6,
    is_etf=False,
    slippage=0.001,
    trail_stop=0.0,        # 0 = 關閉；0.08 = 8% 追蹤停損
)
print(metrics["sharpe_ratio"], metrics["total_return"])

內建台股交易成本

項目計算公式來源
券商手續費成交金額 × 0.1425% × 折扣買賣各一次，最低 20 元（零股 1 元）
證券交易稅賣出金額 × 0.3%股票，僅賣出時收取
ETF 證交稅賣出金額 × 0.1%ETF 優惠
當沖證交稅賣出金額 × 0.15%優惠至 2027 年底
滑價價格 × 0.1%（預設）模擬實際成交差

14 項回傳指標

total_return, annual_return, max_drawdown, sharpe_ratio, sortino_ratio, calmar_ratio, win_rate, profit_factor, total_trades, avg_trade_duration, equity_curve, final_value, 以及逐筆 trades 明細。

記憶體管理：每次 run() 結束後會 del + gc.collect() 釋放 VectorBT Portfolio。 若需重複回測（A/B 比較），請使用 @st.cache_data 包裝結果。

🔔 告警系統

告警系統允許你設定「當某條件成立時通知我」的規則，並在背景或手動觸發掃描。

三種規則類型

📈

價格突破

突破 / 跌破 N 日高低點。參數：direction（high/low）、lookback（5-250 天）。

📊

RSI 閾值

RSI 向上 / 向下穿越設定值。參數：level（10-90）、direction（above/below）。

📡

策略訊號

5 種已驗證策略的當日進場訊號觸發。參數：strategy_key。
